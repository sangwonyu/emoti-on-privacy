# Emoti:on — 공식 소개 & 개인정보 처리방침 페이지

Emoti:on 앱의 GitHub Pages 정적 웹사이트입니다.
순수 HTML + CSS + Vanilla JS로 구현되어 있으며, 빌드 도구 없이 즉시 호스팅됩니다.

## 페이지 구성

- `index.html` — 메인 랜딩 페이지 (앱 소개, 캐릭터, 기능, FAQ, 다운로드 CTA)
- `privacy.html` — 개인정보 처리방침
- `support.html` — 고객 지원 및 자주 묻는 질문
